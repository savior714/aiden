\# \[Project Juwon] PRD (Product Requirements Document)



\## 1. 프로젝트 개요 (Overview)

\* \*\*목적:\*\* 광고와 자극적인 요소가 제거된, 아들 주원이를 위한 순수 학습 및 두뇌 개발용 앱 제작.

\* \*\*목표:\*\* 

&nbsp;   1. 인터넷 연결 없이도 언제 어디서나 실행 가능 (Offline First).

&nbsp;   2. Toy Theater 스타일의 교육용 게임 5종 완벽 이식.

&nbsp;   3. 안드로이드 태블릿(갤럭시탭 S9)에서 네이티브 앱처럼 부드럽게 구동.



\## 2. 사용자 페르소나 및 유저 스토리 (User Personas \& Stories)

\* \*\*주요 사용자:\*\* 이주원 (초등학생 수준의 학습이 필요한 아동)

\* \*\*관리자:\*\* 아빠 (앱 설치 및 난이도 조절 담당)

\* \*\*유저 스토리:\*\* 

&nbsp;   \* "나는 게임을 하다가 갑자기 튀어나오는 광고 때문에 흐름이 끊기지 않기를 원한다."

&nbsp;   \* "나는 수학 문제를 풀거나 미로를 탈출했을 때 즉각적인 칭찬(효과음/애니메이션)을 받고 싶다."



\## 3. 핵심 기능 (Key Features - Scope)

\* \*\*MVP 포함 기능 (구현 순서순):\*\*

&nbsp;   1. \*\*Math Test:\*\* 기본 사칙연산(덧셈, 뺄셈, 곱셈, 나눗셈) 퀴즈. 타이머 모드 및 연습 모드.

&nbsp;   2. \*\*Missing (Math):\*\* 식의 빈칸 채우기 (예: 3 + ? = 10). 역연산 사고력 증진.

&nbsp;   3. \*\*Combo:\*\* 숫자 조합 퍼즐. 특정 수 만들기 등 논리적 사고 게임.

&nbsp;   4. \*\*Maze:\*\* 미로 찾기. 터치 드래그를 이용한 길 찾기 (난이도 조절 가능).

&nbsp;   5. \*\*Merge Fruit:\*\* 물리 엔진 기반 과일 합치기 게임 (Suika Game 스타일). 공간 지각 능력.



\* \*\*제외 범위:\*\* 

&nbsp;   \* 복잡한 서버 기반 랭킹 시스템 (로컬 점수 기록만 유지).

&nbsp;   \* 인앱 결제 및 외부 광고.



\## 4. 사용자 흐름 및 UI/UX 요구사항 (User Flow \& UI/UX)

\* \*\*유저 저니(User Journey):\*\*

&nbsp;   1. 앱 실행 (Splash Screen)

&nbsp;   2. 메인 로비: 5개의 게임 아이콘이 배치된 그리드 화면.

&nbsp;   3. 게임 선택: 아이콘 터치 시 해당 게임 로드 (즉시 실행).

&nbsp;   4. 플레이: 게임 진행 -> 성공/실패 피드백 (시각/청각).

&nbsp;   5. 결과 화면: "참 잘했어요" 메시지 + 재시작/메인으로 가기 버튼.

\* \*\*UI 요구사항:\*\* 

&nbsp;   \* \*\*Large Touch Targets:\*\* 아이 손가락으로도 쉽게 누를 수 있는 큰 버튼.

&nbsp;   \* \*\*Simple Colors:\*\* 눈이 편안하고 직관적인 색상 사용 (Shadcn/UI 기본 테마 활용).

&nbsp;   \* \*\*Landscape Mode:\*\* 태블릿 가로 모드 고정 권장.



\## 5. 성공 지표 (Success Metrics)

\* 앱 실행 후 크래시 없이 5가지 게임이 모두 정상 구동되는가?

\* 터치 반응 속도가 즉각적인가? (딜레이 없음)

\* 갤럭시탭 S9에서 전체 화면으로 꽉 차게 나오는가?



\## 6. 제약 사항 및 고려 사항 (Constraints)

\* \*\*배포:\*\* Play Store 등재 목적이 아닌, 로컬 APK 빌드 및 설치 목적.

\* \*\*리소스:\*\* 이미지, 사운드 등 모든 리소스는 저작권 무료(CC0) 혹은 AI 생성 리소스 사용.
